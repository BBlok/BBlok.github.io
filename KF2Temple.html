Mods - Killing Floor 2

In this project, I decided to challenge myself to make a level for Killing Floor 2 from scratch using the Killing Floor 2 SDK (which is powered by Unreal Engine). Being my first level made using this SDK and my first project working with static meshes and BSP brushes, I decided to start with a relatively simple project and make a level based on the Temple multiplayer level from Goldeneye 007 to apply some of the techniques that I have learned so far in Unreal Engine such as UV map/unwrap, material creation, diffuse and normal maps, and level contruction using a modular set of 3d models. To make this level, I created a series of modular pieces (static meshes) in 3ds Max and later used them to compose it in a modular way. In order to match the mood of the game , I also decided to give the level a dark/sinister feel by manipulating the lighting in the environment.
